# Architecture Contracts Index

本索引用于路由 agent-html architecture contracts。

## Main Chain

```txt
ComponentDeclaration
        ↓
Agent Component Catalog
        ↓
agent-html
        ↓
renderer
        ↓
portable output
```

## Contracts

- `component-source-to-catalog.md`: ComponentDeclaration 到 Agent Component Catalog。
- `catalog-to-agent-html.md`: Agent Component Catalog 到 agent-html。
- `agent-html-to-renderer.md`: agent-html 到 renderer。
- `renderer-to-portable-output.md`: renderer 到 portable output。
