# @agent-html/ahtml

`ahtml` turns semantic `.agent.html` documents into stable, shareable HTML artifacts.

Docs: <https://agent-html.pages.dev/docs>

## Install

```bash
npm install -g @agent-html/ahtml
ahtml
```

## Core Workflow

```bash
ahtml prompt
ahtml build artifact.agent.html
ahtml preview artifact.agent.html
```
