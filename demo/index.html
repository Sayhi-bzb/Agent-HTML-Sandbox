2. 完整 session：功能优先级决策面板

1. User Question

input 
我有 6 个候选功能，要决定下个 sprint 做哪些。
请做一个可视化页面：
- 按影响力/成本展示
- 分组优先级
- 标出依赖关系
- 给我一个可以带回 agent 的决策摘要

2. Agent reads Catalog

base + custom
type CatalogItem 用途 可填信息
config render profile 选择渲染预设 theme, density, tone, width
base card, table, badge, button 基础布局和信息块 title, tone, label
custom priority-matrix 影响力/成本二维图 x, y, items
custom decision-board Now / Next / Later / Cut 分组 columns, items
custom dependency-map 依赖关系图 nodes, edges
custom decision-summary 把人的选择带回 agent format, content

3. Agent Output

agent-html 
<meta-agent theme="neutral" density="compact" tone="dashboard" width="wide" />

<dashboard title="Sprint Priority Decision">
  <priority-matrix x="cost" y="impact">
    <item name="Checkout retry" impact="high" cost="low" />
    <item name="Billing export" impact="high" cost="medium" />
    <item name="Audit log" impact="medium" cost="medium" />
    <item name="Theme editor" impact="medium" cost="high" />
    <item name="Coupon rules" impact="low" cost="medium" />
    <item name="Social sharing" impact="low" cost="high" />
  </priority-matrix>

  <decision-board columns="now,next,later,cut">
    <card column="now" name="Checkout retry" reason="high impact / low cost" />
    <card column="now" name="Billing export" reason="enterprise blocker" />
    <card column="next" name="Audit log" reason="dependency for export" />
    <card column="later" name="Theme editor" reason="useful but expensive" />
    <card column="cut" name="Social sharing" reason="low impact / high cost" />
  </decision-board>

  <dependency-map>
    <edge from="Audit log" to="Billing export" />
    <edge from="Checkout retry" to="Coupon rules" />
  </dependency-map>

  <decision-summary format="prompt" />
</dashboard>

4. parse / validate / sanitize

safety gate
allowed config theme=neutral, density=compact, tone=dashboard, width=wide
allowed blocks dashboard, priority-matrix, decision-board, dependency-map, decision-summary, item, card, edge
allowed attrs title, x, y, impact, cost, column, reason, from, to, format
blocked script, onclick, style, class, unknown tags, full shadcn props, CSS in header

5. SanitizedAgentHtml

checked structure 
Dashboard {
  title: "Sprint Priority Decision",
  children: [
    PriorityMatrix(items x 6),
    DecisionBoard(columns x 4, cards x 5),
    DependencyMap(edges x 2),
    DecisionSummary(format: "prompt")
  ]
}
