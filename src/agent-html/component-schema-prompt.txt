Write agent-html only.

Header:
<meta-agent theme="neutral" density="compact|comfortable" tone="report|dashboard|decision" width="article|dashboard|wide" />

Standard components:
page(title*) -> alert/card/separator/table/list/tabs/accordion/choice-group/slider-control/feedback-box/progress-meter
alert(title? tone?=neutral|danger) -> text
card(title?) -> alert/badge/separator/table/list/tabs/accordion/choice-group/slider-control/feedback-box/progress-meter/text
separator -> none
badge(tone?=neutral|success|warning|danger) -> text
table -> row
row(kind?=header|body) -> cell
cell -> text
list(variant?=ordered|unordered) -> item
item -> text
tabs(default?) -> tab
tab(value* label*) -> alert/card/separator/table/list/accordion/choice-group/slider-control/feedback-box/progress-meter
accordion -> accordion-item
accordion-item(value* title*) -> alert/badge/table/list/text
choice-group(title? mode?=single|multiple default?) -> choice
choice(value* label*) -> text
slider-control(label* value* min? max? step? unit?) -> text
feedback-box(title? placeholder? copy-label?) -> text
progress-meter(label* value* detail?) -> none

Forbidden:
class/className/style/css/Tailwind/shadcn props/Radix props/React props/script/onclick/events/external URLs/unknown tags/unknown attrs.
