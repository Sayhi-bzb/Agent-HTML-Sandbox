# Examples

Check setup before using these examples:

```bash
ahtml status
ahtml doctor
```

## Planning artifact [#planning-artifact]

```html title="artifact.agent.html"
<meta-agent profile="report-default" />

<page title="Implementation Plan">
  <card title="Goal">
    Replace a long Markdown plan with a readable HTML artifact.
  </card>

  <card title="Review Focus">
    <list variant="unordered">
      <item>Confirm the intended behavior before implementation.</item>
      <item>Compare tradeoffs without scanning a wall of text.</item>
      <item>Keep the plan easy to share with reviewers.</item>
    </list>
  </card>
</page>
```

```bash
ahtml validate --input artifact.agent.html
ahtml build --input artifact.agent.html --out dist/html
```

## Code review explainer [#code-review-explainer]

```html title="report.agent.html"
<meta-agent profile="report-default" />

<page title="Streaming Review">
  <card title="Summary">
    The change improves streaming reliability, but reviewers should focus on
    backpressure handling and retry behavior.
  </card>

  <card title="Review Notes">
    <list variant="unordered">
      <item>Check whether slow consumers can block the producer.</item>
      <item>Confirm retry state is reset after a successful stream.</item>
      <item>Keep the explanation shareable outside the Git diff view.</item>
    </list>
  </card>
</page>
```

```bash
ahtml validate --input report.agent.html
ahtml build --input report.agent.html --out dist/html
```

## Decision artifact [#decision-artifact]

```html title="decision.agent.html"
<meta-agent profile="review-dense" />

<page title="Vendor Decision">
  <alert title="Recommendation" tone="neutral">
    Choose Vendor A for the initial rollout.
  </alert>

  <table>
    <row kind="header">
      <cell>Option</cell>
      <cell>Strength</cell>
      <cell>Risk</cell>
    </row>
    <row kind="body">
      <cell>Vendor A</cell>
      <cell>Fastest integration</cell>
      <cell>Higher contract cost</cell>
    </row>
    <row kind="body">
      <cell>Vendor B</cell>
      <cell>Lower price</cell>
      <cell>Longer setup time</cell>
    </row>
  </table>
</page>
```

```bash
ahtml validate --input decision.agent.html
ahtml build --input decision.agent.html --out dist/html
```

## Multi-view artifact [#multi-view-artifact]

```html title="multi-view.agent.html"
<meta-agent profile="ops-compact" />

<page title="Review Workspace">
  <tabs default="summary">
    <tab value="summary" label="Summary">
      <alert title="Decision">Ready to proceed with the current plan.</alert>
    </tab>
    <tab value="details" label="Details">
      <accordion>
        <accordion-item value="risks" title="Risks">
          <list variant="unordered">
            <item
              >Confirm renderer support before exposing new schema
              entries.</item
            >
            <item>Keep custom controls out of the public prompt schema.</item>
          </list>
        </accordion-item>
      </accordion>
    </tab>
  </tabs>
</page>
```

```bash
ahtml validate --input multi-view.agent.html
ahtml build --input multi-view.agent.html --out dist/html
```
